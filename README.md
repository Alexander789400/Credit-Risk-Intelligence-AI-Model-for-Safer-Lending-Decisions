# Credit-Risk-Intelligence-AI-Model-for-Safer-Lending-Decisions
# [DataSet Source](https://www.kaggle.com/datasets/ppb00x/credit-risk-customers)

A comprehensive machine learning project focused on predicting customer credit risk using financial and behavioral data. Features data analysis, preprocessing, model training, and performance evaluation to enhance decision-making in credit scoring and lending.

Used Exploratory Data Analysis techniques, Logistic Regression and Gaussian Naive Bayes.

This dataset classifies people described by a set of attributes as good or bad credit risks.
